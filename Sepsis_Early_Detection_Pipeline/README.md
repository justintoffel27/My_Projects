# Sepsis Early Detection Pipeline

### 1. Download Data
You can download the required dataset ("training_setA") directly from the Google Drive link below.

* **Download Link:** [Google Drive: Sepsis Project Data](https://drive.google.com/drive/folders/14_dAJ7I_yBnm8QdgqbRJ8STEZoQSDX4J?usp=drive_link)
* *(Official Source: [PhysioNet Challenge 2019](https://physionet.org/content/challenge-2019/1.0.0/))*

**Setup Action:**
1. Download the files.
2. Extract them into a folder named `data/` in the project root directory.

### 2. Install Dependencies
Run the following command to install the required Python libraries:

```bash
pip install -r requirements.txt
```
